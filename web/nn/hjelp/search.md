Dersom du vil søke på termar innanfor eit bestemt fagområde, kan du
velje eit :ToolTip{label="domene" title="domene" content-id="domain" }
og eventuelt avgrense søket til bestemte :ToolTip{label="underdomener"
title="underdomene" content-id="subdomain"} i søkemenyen.

Du kan òg avgrense søket til ein bestemt :ToolTip{label="termbase"
title="termbase" content-id="termbase"} ved å veksle frå domenesøk til
termbasesøk og velje basen frå søkemenyen.

I tillegg kan du søke direkte frå termbasesidene, som du finn under
Termbasar i hovudmenyen. 

::HeadingTp{level="h3"}
Søkespråk og målspråk
::

Vel eit søkespråk for å få treff berre på det bestemte språket. Viss
du leitar etter omsetjinga av ein term på eit anna språk, kan du angi
det som målspråk. Då blir både termen på søkespråket og den
tilsvarande :ToolTip{label="tilrådde termen" title="tilrådd term"
content-id="prefLabel"} på målspråket vist i trefflista, viss
tilgjengeleg.

::HeadingTp{level="h3"}
Søketreff og filtrering
::

Tilrådde termar blir markerte med feit skrift i trefflista.

Søketreffa blir sorterte automatisk etter graden av likskap med
søkestrengen. Termar som samsvarar nøyaktig med søkestrengen, står
øvst i trefflista. Deretter kjem treff der søkestrengen er del av ein
:ToolTip{label="fleirordsterm" title="fleirordsterm"
content-id="multiWordTerm"} eller ein :ToolTip{label="samansett term"
title="samansett term" content-id="complexTerm"}.

Du kan filtrere treffa etter språk, domene eller termbase,
:ToolTip{label="termstatus" title="termstatus"
content-id="acceptabilityRating"} og treffmønster ved å trykke på
Filter og velje dei relevante alternativa frå filtermenyen.
