Lisens
::
Termportalen inneheld hovudsakleg
opne data, men nokre av termbasane har restriksjonar for bruk.
Informasjon om kva lisens som gjeld for kvar termbase finn du på
informasjonssida til den aktuelle termbasen.

::HeadingTp{level="h3"}
API
::
Vi arbeider med å utvikle eit eige API for deling av opne data frå Termportalen.

::HeadingTp{level="h3"}
Sitering
::
Dersom du vil bruke ein termpost som kjelde, er det generelle rådet at du lenkar til sjølve termposten og refererer til termbasen den tilhøyrer. Vi anbefaler også å oppgi lesedato, sidan termpostane kan bli endra. Formatet avheng av referansestilen du bruker. Dømet nedanfor er i Harvard-stil og følger formatet for artiklar i elektroniske leksikon utan forfattar.
